## 📋 Описание проекта

Проект по улучшению базовой ML-модели: feature engineering, подбор гиперпараметров и повышение метрик качества. Демонстрирует итеративный подход к улучшению предсказательной модели.

### Бизнес-задача

Повышение качества предсказаний при ограниченных ресурсах. Улучшение baseline позволяет получать большую бизнес-ценность без усложнения инфраструктуры.

**Бизнес-ценность:**
- Рост точности предсказаний
- Снижение количества ложных срабатываний
- Более обоснованные решения на основе модели

### Задача машинного обучения

**Тип задачи:** Классификация / регрессия (в зависимости от домена)

**Метрики качества:**
- Улучшение F1 / Accuracy / AUC по сравнению с baseline
- Сбалансированность precision и recall

## 🎯 Ключевые достижения

- ✅ Улучшение метрик относительно исходного baseline
- ✅ Систематический feature engineering
- ✅ Подбор гиперпараметров (Optuna / GridSearch)
- ✅ Воспроизводимость экспериментов (MLflow)
