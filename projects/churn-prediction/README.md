# 🏠 Предсказание оттока клиентов

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![Status](https://img.shields.io/badge/Status-Production-success.svg)]()

## 📋 Описание проекта

Система предсказания оттока клиентов для Яндекс Недвижимости с полной автоматизацией процесса обработки данных и обучения модели. Проект демонстрирует end-to-end ML-пайплайн от сбора данных до деплоя в продакшн.

### Бизнес-задача

Яндекс Недвижимость теряет клиентов из-за разногласий по цене на объекты недвижимости. Задача — создать систему предсказания оттока клиентов, которая поможет менеджерам заранее выявлять рискованных клиентов и принимать превентивные меры.

**Бизнес-ценность:**
- Увеличение среднемесячного количества сделок
- Снижение потерь клиентов
- Оптимизация работы менеджеров

### Задача машинного обучения

**Тип задачи:** Бинарная классификация (отток / не отток)

**Целевая переменная:** Признак оттока клиента

**Метрики качества:**
- F1-score: **0.84** (основная метрика)
- Accuracy: **0.87**
- Precision: **0.89**
- Recall: **0.82**

## 🎯 Ключевые достижения

## 🛠 Технологический стек

### Основные технологии
- **Python** 3.8+
- **Scikit-learn** — обучение моделей
- **Pandas, NumPy** — обработка данных
- **Matplotlib, Seaborn** — визуализация

### MLOps инструменты

### Инфраструктура
- **PostgreSQL** — база данных

## 🏗 Архитектура решения

```
1. Сбор данных (Airflow DAG)
   ↓
2. Очистка и обработка данных (Airflow DAG)
   ↓
3. Feature Engineering (DVC pipeline)
   ↓
4. Обучение модели (DVC pipeline + MLflow)
   ↓
5. Сохранение модели в S3
   ↓
6. Версионирование через DVC и MLflow
```

## 🚀 Быстрый старт

1. Клонируйте репозиторий: `git clone https://github.com/DataClasse/churn-prediction.git`
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте `.env` (БД, S3)
4. Запустите Airflow DAGs, затем `dvc repro`
