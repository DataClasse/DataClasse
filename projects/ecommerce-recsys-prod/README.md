## 📋 Описание проекта

Проект демонстрирует **полный цикл работы с ML-моделью**: от постановки задачи и сбора данных до разработки рекомендательной системы и выката в продакшен. Реализована гибридная рекомендательная система (коллаборативная фильтрация + контентные методы) с REST API, контейнеризацией и пайплайном доставки.

### Бизнес-задача

Персонализированные рекомендации для пользователей с целью увеличения вовлеченности и конверсии. Система должна работать в production-режиме с приемлемой задержкой и масштабируемостью.

**Бизнес-ценность:**
- Рост времени использования сервиса
- Улучшение конверсии
- Стабильная работа в продакшене

### Задача машинного обучения

**Тип задачи:** Рекомендательная система (Ranking), полный цикл: данные → модель → API → деплой.

**Метрики качества:**
- NDCG@10, Precision@10, Recall@10
- Время ответа API, доступность сервиса

## 🎯 Ключевые достижения

- ✅ **Полный цикл ML:** постановка задачи → сбор и обработка данных → обучение модели → выкат в продакшен
- ✅ **Рекомендательная система:** коллаборативная фильтрация (Implicit) + контентные признаки (CatBoost), гибридная схема
- ✅ **Production:** REST API (FastAPI), Docker, при необходимости CI/CD и мониторинг
- ✅ **Воспроизводимость:** версионирование данных, моделей и артефактов

## 🛠 Технологический стек

- **Python** 3.8+, **Implicit**, **CatBoost**, **Pandas**, **NumPy**
- **FastAPI** — REST API
- **Docker** — контейнеризация
- **MLflow** — эксперименты и артефакты
- **DVC** (опционально) — версионирование данных

## 🏗 Архитектура решения

```
1. Постановка задачи и сбор данных
2. Обработка данных и feature engineering
3. Обучение моделей (CF + content-based)
4. Гибридная модель и оценка (NDCG@10 и др.)
5. Упаковка в API (FastAPI)
6. Выкат в продакшен (Docker, при необходимости K8s)
7. Мониторинг и поддержка
```
