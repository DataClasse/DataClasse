# 🎵 Рекомендательная система

## 📋 Описание проекта

Гибридная рекомендательная система для музыкального стриминга, объединяющая коллаборативную фильтрацию и контентные методы. Система развернута как микросервис с REST API и интегрирована в веб-приложение (Яндекс Музыка).

### Бизнес-задача

Увеличение вовлеченности пользователей музыкального стриминга через персонализированные рекомендации. Система должна работать как для существующих пользователей, так и для новых (холодный старт).

**Бизнес-ценность:**
- Увеличение времени прослушивания
- Повышение конверсии в подписку
- Улучшение пользовательского опыта

### Задача машинного обучения

**Тип задачи:** Рекомендательная система (Ranking)

**Метрики качества:**
- **NDCG@10: 0.73** (основная метрика)
- Precision@10: 0.68
- Recall@10: 0.75

## 🎯 Ключевые достижения

- ✅ **NDCG@10 = 0.73** — высокое качество рекомендаций
- ✅ **Гибридный подход** — комбинация коллаборативной фильтрации и контентных методов
- ✅ **REST API** — полноценный микросервис с документацией
- ✅ **Docker deployment** — контейнеризация для легкого развертывания

## 🛠 Технологический стек

### Основные технологии
- **Python** 3.8+
- **Implicit** — коллаборативная фильтрация (ALS, BPR)
- **CatBoost** — градиентный бустинг для контентных признаков
- **Pandas, NumPy**, **Scikit-learn**

### MLOps инструменты
- **FastAPI** — REST API
- **Docker** — контейнеризация
- **Pydantic**, **Swagger/OpenAPI**

### Инфраструктура
- **PostgreSQL**, **Redis** (опционально)

## 📊 Метрики качества

| Метрика | Значение | Описание |
|---------|----------|----------|
| **NDCG@10** | 0.73 | Normalized Discounted Cumulative Gain (основная) |
| **Precision@10** | 0.68 | Точность топ-10 рекомендаций |
| **Recall@10** | 0.75 | Полнота покрытия релевантных треков |

## 🏗 Архитектура решения

```
Web App → FastAPI (Recommendation) → Implicit (CF) + CatBoost (Content) → Hybrid → Top-K
```
